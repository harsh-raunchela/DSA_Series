# 867. Transpose Matrix

> Easy | Arrays | Matrix | Simulation

---

## 📝 Problem Description

Given a **2D integer array** `matrix`, return its **transpose**.

👉 In a transpose of a matrix:
- The **rows** become **columns**.
- The **columns** become **rows**.

Basically, you flip the matrix over its **main diagonal**.

---

## 📈 Examples

### Example 1:

Input: matrix = [[1,2,3], [4,5,6], [7,8,9]]

Output: [[1,4,7], [2,5,8], [3,6,9]]
✅ Rows turned into columns!

---

### Example 2:

Input: matrix = [[1,2,3], [4,5,6]]

Output: [[1,4], [2,5], [3,6]]
